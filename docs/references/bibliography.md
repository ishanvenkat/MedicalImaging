- Expanded educational dataset: ~9,223 X-ray images (JPG/JPEG/PNG)
- Layout: `data/train/` (~8,723) and `data/test/` (500 held-out)
- Classes: `fractured`, `not fractured`
- Original 90-image Colab set archived under `data/archive/`

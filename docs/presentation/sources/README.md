# GHLC Official Presentation Sources

Authoritative program materials for the 2026 Student Speaker Series (SSS). Derived specs and build guides in this repo should match these files.

| File | Purpose |
|------|---------|
| [`GHLC_Presentation_Guidelines.pdf`](GHLC_Presentation_Guidelines.pdf) | Application process, timing, recording/submission, required slide sections |
| [`2026 Student Speaker Series Slide Template A.pdf`](2026%20Student%20Speaker%20Series%20Slide%20Template%20A.pdf) | Mandatory slide template layout (title → intro → content → references → acknowledgements → thank you → contact) |

## Key Requirements (summary)

- **Duration:** 8 minutes or less (spoken content)
- **Deadline:** July 20, 2026, 11:59 PM Eastern Time
- **Template:** Must use the 2026 GHLC Student Speaker Series slide template — copy to a folder you own before editing
- **Submission:** Pre-recorded Zoom video (camera and mic on); upload to Google Drive folder `LastName_FirstName_TITLE_OF_PRESENTATION` with editor access; submit via [application form](https://forms.gle/Wx7L1WS59DECbtAV8)
- **Recording walkthrough:** [YouTube playlist](https://www.youtube.com/playlist?list=PLD4D2lXEsA-WC-xvwD8DEXiUi_k657eZz)
